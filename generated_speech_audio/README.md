# Generated Speech Audio

В этой папке хранятся аудиофайлы, сгенерированные с помощью TTS, которые в дальнейшем используются голосовым ассистентом.


## Назначение

Аудиофайлы, хранящиеся здесь, являются промежуточными или конечными результатами синтеза речи.


В дальнейшем они используются в качестве входных данных для:

- генерации синхронизации губ
- рендеринга видео с говорящей головой



## Очистка

Вы можете безопасно удалить все файлы в этом каталоге в любое время.
Они будут сгенерированы заново при следующем запуске.
