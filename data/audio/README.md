\# Audio Voice Samples

Здесь хранятся экземпляры голосов, используемые для создания голосовых клонов персонажей.

Каждая персона имеет собственную директорию с набором аудиофайлов, на основе которых модель обучается воспроизводить голос.

Логика обработки реализована в:

src/voice\_clone/clone\_manager.py


## Структура хранения:

├── audio/

│ ├── Alex/

│ │ ├── voice\_1.wav

│ │ ├── voice\_2.ogg



\## Формат:

talking-head-assistant/data/audio/<person\_name>/<any\_name>.(wav|ogg)
