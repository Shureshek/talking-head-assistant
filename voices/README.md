# Voices

В этой папке хранятся обработанные из сэмплов клоны голосов, используемые в talking-head assistant.



## Назначение



Файлы в этой директории представляют собой \*\*готовые к использованию модели голосов / эмбеддинги\*\*.  

Они загружаются напрямую во время работы ассистента для:



- синтеза речи (TTS)  

- генерации аудио с конкретным голосом  

- вывода речи для выбранного спикера



## Примечания

Если папка пустая или файлы удалены, клоны голосов можно восстановить автоматически (при наличии исходного аудио)
